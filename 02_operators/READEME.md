# ➕ Python Operators

Welcome to the **Operators** section of my **Python Practice** repository.

This folder covers all fundamental Python operators used to perform arithmetic calculations, comparisons, logical operations, and other common programming tasks.

Understanding operators is essential because they are used in almost every Python program.

---

* Arithmetic Operators
* Assignment Operators
* Comparison Operators
* Logical Operators
* Membership Operators

## 📌 Operators Included

| Operator Type | Description                                                   |                          |
| ------------- | ------------------------------------------------------------- | ------------------------ |
| Arithmetic    | Mathematical operations (`+`, `-`, `*`, `/`, `//`, `%`, `**`) |                          |
| Assignment    | Assign and update values (`=`, `+=`, `-=`, `*=`, etc.)        |                          |
| Comparison    | Compare two values (`==`, `!=`, `>`, `<`, `>=`, `<=`)         |                          |
| Logical       | Combine conditions (`and`, `or`, `not`)                       |                          |
| Membership    | Check membership (`in`, `not in`)                             |                          |
| Identity      | Compare object identity (`is`, `is not`)                      |                          |
| Bitwise       | Perform binary operations (`&`, `                             | `, `^`, `~`, `<<`, `>>`) |
